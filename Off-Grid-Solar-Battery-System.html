Off-Grid Solar Battery System

12 x SunPower BLK 250W Solar Panel = 3kW

These panels initially came with micro-inverters but they have been removed. The labels show specs for AC but the panels output DC. They were manufactured between 2014 and 2017. The panels are in a 3S3P configuration on the West side and a 3S1P array on the East. The solar panels original specification was 20.5% efficiency. Mounted on Ironridge XR-100 rail system.

Maximum system voltage: 600V UL

Charge controllers

The East controller is an powMR 50A MPPT, the West unit is a 60A version.

Battery 4kWh 48V

The battery system consists of two Chevy Volt 2kWh 48V modules in parallel

Inverters

The primary Off-Grid inverter is a 4kW low frequency inverter with 85% efficiency. A 1kW grid tie inverter can run in parallel with the Off-Grid inverter. A 400W 12V inverter is ran with a step down converter from the 48V battery bus when only small loads are demanded.

Control system

The various sensors and circuits are connected to an Arduino Mega2560 with a protoboard containing several voltage dividers. The arduino is connected to a Raspberry pi acting as a data server for the output of the Arduino serail port into a python script that recieves a .csv and has two outputs it prints a constrained set of data to a terminal window for live monitoring of the system and it writes a line to a .csv file with many measurements and culculations of the system performance.
